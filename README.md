## 🚀 Projects

<h3 align="left">
  ⭐ V2X based Cooperative Perception &nbsp;
  <a href="https://github.com/Pavankumarsp02/late_fusion_yolos_cpp.git">

* Developed a V2X based perception system using YOLOs-Cpp.
* Implemented the complete system logic and node communication using ROS2 Jazzy.
* Generated a comprehensive panoramic view (Unified list) by merging detections from multiple camera sensors.
* Utilised RViz2 to provide real time visualization of fused sensor data.
* Evaluated detection performance and system reliability in complex environmental scenarios.
* Developed in Linux (Ubuntu 24.04) and utilised docker for efficient deployment of model.

⭐ **Automatic Vehicle Speed Control using Bluetooth Low energy (BLE)**
* Designed & Prototyped **Custom PCB board** integrating an **ESP32** µcontroller & on-board DC-DC power regulation.
* Developed a custom Lithium-ion battery pack for optimized portable energy management.
* Implemented multiple modules assigned with specific speed values for localized environment control.
* Automated prototyped car speed adjustment based on the nearest BLE speed module.
* Engineered precise speed regulation via **Pulse Width Modulation** signals to the motor driver.
* Real-time software logic to ensure car speed remains less than or equal to the detected module limit.

⭐ **Geometric Optimisation of Interconnection Layers in IGBT/Diode**
* Performed experimental validation of silver sintering layers to optimize semiconductor package reliability.
* Developed automated simulation workflows to predict material behavior using **Ansys Mechanical**.
* Optimized material usage by **59.55%** through structured DOE (Design of Experiments) and simulation.
* Validated model against peer-reviewed literature to ensure predictive accuracy and reliability.

⭐ **Smart Shopping Cart with Weight Verification**
* Integrated barcode scanner for rapid product entry and digital cart management.
* Real time database cross-check compares physical cart weight with item specifications.
* Intuitive physical button (Red) for quantity adjustment and easy deletion of the last item.
* Dual button trigger (Red & Green) combined with vendor RFID authorization for final validation.
* Generates a dynamic QR code with the total amount for direct digital UPI transactions.
* System blocks payment and displays alert if physical weight mismatches the database.

⭐ **CRC Detection**
* Designed a high-speed **Verilog hardware logic** in **Cadence** to implement robust CRC algorithm for data integrity.
* Integrated real-time error detection module into digital communication architecture to flag transmission faults.
* Verified hardware accuracy via logic analyzer testing to ensure reliable, high speed data processing.
